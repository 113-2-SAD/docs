## 環境準備

為了順利進行課程，請確保已準備以下環境：

1. **電腦**：Windows、Mac、Linux 皆可
2. **開發工具**：Visual Studio Code（或任何你習慣的 Code Editor、IDE）、終端機 (Terminal)
3. **瀏覽器**：Chrome、Firefox、Edge 皆可
4. **帳號**：GitHub
5. **Windows 用戶建議**：安裝 **WSL 2 (Windows Subsystem for Linux 2)**，以免在後續課程中遇到問題
6. **安裝 Node.js**：[Node.js 下載連結](https://nodejs.org/)

---

## 安裝 pnpm

**pnpm** 是一款現代化、快速且高效的 Node.js 套件管理工具。本課程將使用 pnpm 來管理依賴（Dependencies）。

### 安裝指令：

```bash
npm install -g pnpm
```

> **為什麼選擇 pnpm？**
>
> - 速度更快，安裝時不會重複下載相同套件，也節省磁碟空間
> - 兼容 npm，大部分指令相同
> - 與現代的專案架構（如 monorepo）高度相容

安裝完成後，你可以使用以下指令來確認是否安裝成功：

```bash
pnpm -v
```
